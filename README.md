CodeCopy
=====

用python语言写的代码复制工具程序，可用于软著著作权源程序代码的复制或其他需要复制项目工程代码的地方

作用：  
- 复制项目工程代码到一个Word文档中  
- 生成一个项目工程文件（文件树状图）  
- 可自定义筛选条件和文档参数  
- 轻松生成文件目录

使用
--------
下载CodeCopy.py，运行即可
